# AI Agent 学习项目

## 项目简介
个人 AI Agent 学习项目，按阶段（week/stage）组织，涵盖 RAG、LangChain、LangSmith、MCP 协议等主题。

## 目录结构
- `week1~6/` — 早期学习阶段
- `stage2/` — 第二阶段
- `stage3/` — 当前阶段，包含 RAG、LangSmith、MCP 等模块
- `fastapi-basics/` — FastAPI 相关练习

## 技术栈
- Python 3.x
- LangChain / LangGraph
- LangSmith（追踪调试）
- MCP 协议
- FastAPI

## 约定
- 代码注释和回复均使用中文
- 每个 stage 下有独立的 `venv` 虚拟环境
